Websites

This is the back-end screen where you can view a list with all the websites from your gallery. Here you can approve any new website (select the website and click on "Approve" button).

Any new website which is waiting for your approval to be listed into the gallery will appear first and on status column will be display "Pending" with red.

If you choose to "Notify user when the website was approved" from Options, than an email will be sent with a notification that the website has been approved.

If you wish one website to be listed before all the others you can make him featured. By default a website is not featured.

If you wish one website not to be listed at all, you can unpublish him. By default a website is published.
